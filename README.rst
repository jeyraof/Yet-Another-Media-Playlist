YAMP_
=====

YAMP (Yet Another Media Player) is open-sourced playlist system.
This is developed for archiving unlimited items of various media service such as Youtube_, Soundcloud_ and etc.

.. _YAMP: http://yamp.monodiary.net/#/
.. _Youtube: http://youtube.com/
.. _Soundcloud: http://soundcloud.com/
